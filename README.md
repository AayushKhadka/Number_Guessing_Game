# Number Guessing Game

A simple console-based number guessing game written in Python. The player selects a difficulty level, then tries to guess a randomly generated number within a limited number of attempts. Points are deducted for each incorrect guess.

## Features

- Three difficulty levels: Easy, Medium, Hard.
- Random number generation with different ranges and attempt counts.
- Score tracking with penalties for wrong guesses.
- Colorful console output using the `colorama` library.

## Requirements

- Python 3.6 or later
- `colorama` package

Install dependencies with pip:

```bash
pip install colorama
```

## Usage

Run the game with Python:

```bash
python Number_Gussing_Game.py
```

Follow the on-screen prompts to select a difficulty level and start guessing. Your score will be displayed at the end.

## Code Standard Guidelines

The code is written in a clear, procedural style with comments at the top describing the purpose, author, and basic behavior. It uses meaningful variable names and consistent indentation. Input validation is simple but handles invalid difficulty choices by defaulting to Medium.
